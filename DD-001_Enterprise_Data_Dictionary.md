# Purpose

The Enterprise Data Dictionary defines the official vocabulary of the Operational Governance System (OGS).

Every governance document, register, report, evidence item and AI inventory shall use the definitions contained in this dictionary.

This document is the Single Source of Truth (SSOT) for business terminology.

---

# Core Business Entities

| Entity | Definition |
|---------|------------|
| Company | Legal entity implementing the Operational Governance System |
| Person | Individual participating in governance activities |
| Organization | Internal or external organization |
| Role | Governance responsibility assigned to a person or organization |
| AI System | AI system used, developed, integrated or monitored |
| AI Use Case | Business process using one or more AI systems |
| Document | Controlled governance artifact |
| Evidence | Verifiable information supporting compliance |
| Requirement | Legal, regulatory or internal obligation |
| Control | Measure implemented to satisfy one or more requirements |
| Risk | Potential event affecting governance objectives |
| Policy | Approved governance rule |
| Audit | Independent assessment of compliance |
| Inspection | External verification activity |
| Repository | Version-controlled document repository |
| Hash | SHA-256 integrity value |
| OTS | OpenTimestamps proof |
| Report | Governance output document |
